Reads the input RPM of the gearbox.

Parameters
destDestination RPM reading pointer
check_sanityShould this function compare N2 to N3? This is a sanity check done in gears 2,3 and 4, where N2 and N3 must be roughly the same value (+/-50RPM). If that is not the case, then something is wrong with the gearbox input speed sensors
Returns
true If sensor reading OK
false If sensor reading failed, or if sanity check failed
