void Solenoid::write_pwm_12_bit ( uint16_t  pwm_raw,
bool  voltage_compensate = true 
)

Writes a 12-bit PWM duty to the solenoid.

Parameters
percentRequested PWM duty (0-4096)
voltage_compensate(Default true) - Tells the solenoid API weather to take into account the real voltage being supplied to the solenoid in order to maintain a constant current.

For instance, if the desired PWM is based on a 12.0V voltage, but the actual voltage being supplied to the solenoid is 14.0V, then a ~16% reduction in PWM will be made in order to keep the current being sent to the solenoid the same
